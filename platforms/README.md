# Plataformas

Use as páginas da plataforma para entender a origem da telemetria antes de chegar a Sentinel.

## Sequência de conclusão
1. Leia as páginas do sistema operacional para entender o comportamento do endpoint e do dispositivo.
2. Leia as páginas da plataforma em nuvem para compreender o plano de controle, a postura, a identidade e o comportamento dos recursos.
3. Mapeie cada plataforma para as tabelas Sentinel e buscas que dependem dela.
4. Registre as lacunas de cobertura onde existe uma plataforma, mas Sentinel ainda não recebe a telemetria correta.

## Como usar essas páginas
- Aprenda como funciona a plataforma a nível de caça.
- Revise quais atividades de segurança são importantes para a plataforma.
- Confirme quais tabelas Sentinel representam a plataforma.
- Identifique ideias de busca, candidatos a casos de uso e telemetria ausente.
