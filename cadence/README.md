# Cadência de Caça

Esta seção define o ritmo operacional do processo de caça a ameaças.

## Cadências
- [Daily Hunt](daily-hunt.md): revisão curta e repetível para risco ativo e integridade dos dados.
- [Busca semanal](weekly-hunt.md): revisão mais profunda de padrões e busca de linha de base.
- [Intel Hunt](intel-hunt.md): caças orientadas por eventos de IOCs, TTPs, CVEs ou inteligência de fornecedor.
- [Caça Mensal](monthly-hunt.md): cobertura, qualidade de detecção, custo e revisão estratégica de caça.
- [Hunt Calendar](hunt-calendar.md): calendário recorrente para planejamento e rotação.
