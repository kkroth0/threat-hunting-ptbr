# Pesquisa Externa

Páginas de pesquisa externa mapeiam ideias públicas de caça a ameaças nesta biblioteca local centrada em Sentinel.

## Fontes
- [Mapeamento de chamas HEARTH](hearth/README.md): hipóteses de caça HEARTH mapeadas para tabelas Sentinel, candidatos KQL, candidatos de caso de uso, cadência, ferramentas e relevância LATAM.

## Usage
Use esta seção como uma camada de entrada. Mantenha a pesquisa original vinculada e, em seguida, converta ideias validadas em páginas de busca locais, arquivos KQL e casos de uso.
