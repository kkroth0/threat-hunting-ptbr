## Consulta inicial
```kql
NetskopeAlerts_CL
| where TimeGenerated >= ago(24h)
| summarize Events = count() by bin(TimeGenerated, 1h)
| order by TimeGenerated desc
```

## Projeção Rápida
```kql
NetskopeAlerts_CL
| where TimeGenerated >= ago(24h)
| project account_id, account_name, action, activity, alert, alert_id, alert_name, alert_type
| take 100
```
