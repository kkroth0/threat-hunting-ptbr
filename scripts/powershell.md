# PowerShell

Windows scripts auxiliares de coleta, enriquecimento, triagem e resposta de endpoint.

## Uso da biblioteca
- Armazene ajudantes, coletores, analisadores ou fluxos de trabalho de enriquecimento reutilizáveis aqui.
- Vincule scripts de volta à página de busca que os utiliza.
- Prefira Sentinel/KQL para lógica de caça repetível e scripts para enriquecimento ou lacunas de coleta.
