# SecurityEvent

## Objetivo de caça
Windows Logs de eventos de segurança coletados em Sentinel, úteis para buscas de logon, processo, conta, serviço e política.

## Consulta inicial
```kql
SecurityEvent
| where TimeGenerated >= ago(24h)
| summarize Events = count() by bin(TimeGenerated, 1h)
| order by TimeGenerated desc
```

## Projeção Rápida
```kql
SecurityEvent
| where TimeGenerated >= ago(24h)
| project Account, AccountDomain, AccountExpires, AccountName, AccountSessionIdentifier, AccountType, Activity, CACertificateHash
| take 100
```
