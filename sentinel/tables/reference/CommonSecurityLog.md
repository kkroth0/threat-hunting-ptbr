## Consulta inicial
```kql
CommonSecurityLog
| where TimeGenerated >= ago(24h)
| summarize Events = count() by bin(TimeGenerated, 1h)
| order by TimeGenerated desc
```

## Projeção Rápida
```kql
CommonSecurityLog
| where TimeGenerated >= ago(24h)
| project Activity, CollectorHostName, Computer, DestinationDnsDomain, DestinationHostName, DestinationIP, DestinationNTDomain, DestinationProcessId
| take 100
```
