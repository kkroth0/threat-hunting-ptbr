# HEARTH Rastreador de cobertura

Este rastreador mostra a maturidade atual da engenharia de caça para cada chama HEARTH nesta biblioteca Sentinel.

O status de cobertura significa que um iniciador KQL existe e está mapeado para tabelas Sentinel disponíveis. Isso não significa que a consulta foi testada ou promovida para uma regra analítica de produção.
