# Ferramentas

As ferramentas são documentadas pelas tabelas Sentinel que alimentam, pelas buscas que suportam e pelas perguntas de segurança que ajudam a responder.

## Sequência de conclusão
1. Comece com as ferramentas que já alimentam as tabelas Sentinel.
2. Aprenda como cada ferramenta funciona e quais atividades de segurança ela pode detectar.
3. Revise as tabelas Sentinel mapeadas e o uso de busca de chaves.
4. Converta buscas úteis repetidas em casos de uso ou candidatos à detecção.
5. Rastreie ferramentas sem tabela Sentinel confirmada como lacunas de integração ou normalização.

## Como usar essas páginas
- Use as seções de finalidade e fluxo de trabalho para entender a ferramenta.
- Use as seções de revisão para saber o que inspecionar durante as caçadas diárias, semanais, de informações e mensais.
- Use a lista da tabela Sentinel para encontrar os dados que respaldam cada busca.
- Utilize a lista de verificação de maturidade para melhorar a cobertura ao longo do tempo.
