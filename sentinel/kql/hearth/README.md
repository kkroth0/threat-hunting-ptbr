# HEARTH KQL Iniciantes

Esta pasta contém um candidato inicial KQL gerado por chama HEARTH.

Esses arquivos são baseados nas tabelas Sentinel atualmente mapeadas neste repositório. Elas não são detecções de produção até que sejam testadas em seu espaço de trabalho Sentinel.

## Escopo
- Arquivos KQL iniciais dedicados: 221
- Fila de prioridade: P1 (25), P2 (114), P3 (76), P4 (6)
- Status de cobertura: Starter KQL - validar em Sentinel (198), Starter - validar telemetria SaaS (7), Starter - validar telemetria Linux (7), Parcial - ajuste personalizado necessário (5), Starter - validar telemetria de nuvem (4)

## Como usar
1. Comece com buscas P1 e P2.
2. Confirme se as tabelas referenciadas estão ingerindo dados.
3. Execute a consulta com o lookback padrão.
4. Ajuste termos, campos, junções, limites e listas de permissões.
5. Promova lógica estável em um caso de uso ou regra analítica.
